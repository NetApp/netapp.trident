Release Notes: Ansible Validated Content for NetApp Trident

v1.0.0
======

Release Summary
---------------

This is the first major release of the ``netapp.trident`` collection.

New Roles
---------

- netapp.trident.create_k8s_objects - Create Kubernetes objects (StorageClasses, PVCs, VolumeSnapshots) using NetApp Trident
- netapp.trident.delete_k8s_objects - Delete Kubernetes objects (StorageClasses, PVCs, VolumeSnapshots) created by NetApp Trident
